# CLAUDE.md - FleurFinder AI Guidelines

## Project Overview
FleurFinder is a digital catalog for "Language of Flowers" (Meaningful Flowers). 
It helps users find the perfect preserved flower based on emotional messages.

## Tech Stack
- Framework: Next.js 15 (App Router)
- Styling: Tailwind CSS
- Language: TypeScript
- Database: Supabase (Planned)

## Development Rules
1. **Plan Mode First**: AI must provide a plan and get approval before writing any code.
2. **Atomic Tasks**: Break down features into small tasks in TASKS.md.
3. **Clean Code**: Use functional components and follow Next.js 15 best practices.
4. **Git Discipline**: Recommend a commit message after every task completion.

## Design Guidelines
- Aesthetic: Soft, premium, and meaningful.
- Spacing: Consistent padding/margin using Tailwind (p-4, m-8, etc.).
- Images: Use next/image with optimized settings.
